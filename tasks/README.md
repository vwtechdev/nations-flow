# Tasks - Tarefas Futuras

Esta pasta contém documentação de implementações futuras e melhorias planejadas para o projeto Nations Flow.

## Tarefas Documentadas

### 1. Seleção Múltipla nos Filtros
**Arquivo:** `selecao-multipla-filtros.md`

Implementação de seleção múltipla nos filtros usando Select2, permitindo que usuários selecionem mais de um valor simultaneamente nos filtros de:
- Categorias
- Campos
- Pastores
- Igrejas
- Usuários

**Status:** Planejado  
**Prioridade:** Média  
**Complexidade:** Média

### 2. Histórico e Informações Completas de Pastores
**Arquivo:** `historico-pastores.md`

Implementação de sistema completo de gestão de informações de pastores, incluindo:

- Familiares (nome e idade)

**Status:** Planejado

**Complexidade:** Alta

## Como Adicionar Novas Tarefas

1. Crie um arquivo `.md` na pasta `tasks/` com o nome descritivo da tarefa
2. Use o formato markdown para documentar:
   - Visão geral
   - Objetivo
   - Alterações necessárias
   - Arquivos afetados
   - Ordem de implementação
   - Testes necessários
3. Atualize este README com a nova tarefa

## Convenções

- Use português para documentação
- Seja específico sobre arquivos e funções
- Inclua exemplos de código quando relevante
